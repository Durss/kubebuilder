What is the purpose of this application?

  • If someday the administrators want to add a new kube to the game, they can watch the most wanted kubes.
    That said, the kubes that are on the podium may have more chance to be added to the game.

    In no case, creating a kube on this application will cause any automatic update of the game !
    The administrators are free to use this list or not.

Votes and creations :

  • You can only vote 3 times a day and once for the same kube.
  • The votes are weighted according to the statistics of the player. Are taken into account, the number of discovered zones and the number of construction points. The more points a player has, the more his vote will count facing the other players.
  • To prevent from abuse, it is possible to create only 3 kubes a day.

Banishment :

  • If you abuse of the vote, creation or report rights, all of those will be removed without notice and without possible negotiation.
